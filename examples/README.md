# Ejemplos de configuración

Este directorio contiene ejemplos de cómo usar el sistema ELT.

## Archivos

- `usage_examples.py`: Ejemplos de uso programático del sistema

## Ejecutar ejemplos

```bash
# Asegúrate de que el entorno virtual esté activado
source venv/bin/activate

# Edita usage_examples.py y descomenta el ejemplo que quieras ejecutar
python examples/usage_examples.py
```
